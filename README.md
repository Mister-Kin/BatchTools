# Batch Tools

## Introduction
Some batch scripts for easily doing work.

## Features
- ffmpeg.sh
  - 给图片添加版权水印并压缩
  - 合并视频和音频：mp4+m4a/mp3
  - 生成avc编码的mp4格式视频（libx264）
  - 压缩图片（原格式压缩或者转换为webp格式）
  - 压缩视频，转换为hevc编码的mp4格式（libx265）
  - 重命名音频
  - 为音频添加封面图
  - 获取音频封面图
  - webp格式转png格式（删除源文件）
  - tga格式转png格式
  - 无损分割视频
  - flv格式转mp4格式
  - 显卡加速将图片序列合成为视频（不再维护该功能）
- text.sh
  - 合并文本文件
  - 合并文本文件（逐行合并）
